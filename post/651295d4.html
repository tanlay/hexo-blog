pyenv安装

发表于 2019-07-26 更新于 2020-01-13 分类于 Python 阅读次数：

Pyenv是多版本Python管理器,可以同时管理多个Python版本共存, 区别于virtualenv.

安装

基本过程是官网下载源码包然后安装, 需要安装gcc, 等库的依赖安装.

查看支持的版本

pyenv install –list

相关注意事项

因各种原因, 下载速度可能只有几KB, 甚至超时, 可将下载地址替换成国内镜像后在下载.
